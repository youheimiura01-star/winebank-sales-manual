## 1.2 ビジネスモデルの全体像

### サービスの流れ（4ステップ）

```
① 購入  →  ② 保管  →  ③ 運用  →  ④ 売却 or 飲用
   ↓          ↓          ↓           ↓
コンシェルジュが   専門倉庫で    アプリで      マーケットプレイスで
最適なワインを    最適環境管理   リアルタイム    売却、または
ポートフォリオ              価格追跡      お好きなタイミングで
構築                               配送・飲用
```

**ポイント**: 「投資として値上がり益を狙える」+「飲んで楽しむこともできる」= **二刀流の資産**

### 収益構造
- ワインの値上がり益がお客様の利益
- WineBankの手数料: 売却価格の **10%**（売却時のみ発生）
- 保管中の手数料は各プランに含まれる

## 1.6 よくある質問（社内向け）

**Q: ワインは腐らないのですか？**
> A: ファインワインは適切な温度・湿度管理のもとで長期保管すると、むしろ熟成により価値が上がります。WineBankの専門倉庫で最適環境を維持しています。保険・補償制度もあります。

**Q: 本当に売れるのですか？（換金性）**
> A: WineBankのマーケットプレイスで売却可能です。また、コンシェルジュが売却のサポートもします。世界的にファインワイン市場は拡大しており、流動性は確保されています。

**Q: 株やFXと何が違うのですか？**
> A: ワインは「実物資産」です。株式市場との相関が低く、分散投資に最適です。さらに、値下がりしても「飲んで楽しめる」という他の投資にはないメリットがあります。

**Q: なぜ値上がりするのですか？**
> A: ファインワインは①生産量が限られている、②飲めば消費されるため供給が減り続ける、③世界的に需要が増加している、という3つの理由で価格が上昇しやすい構造にあります。

**Q: 法人で購入する場合、経費になりますか？**
> A: 接待目的での購入は接待交際費として経費計上できる可能性があります。ただし、具体的な税務処理はお客様の顧問税理士にご確認いただくようご案内ください。

**Q: 怪しい商品ではないですか？**
> A: WineBankは金融・不動産・広告・倉庫業界の大手企業から出資を受けており、五大シャトーの正規代理店実績もあります。保有ワインには保険・補償制度が適用されます。海外では富裕層の間で20年以上前から一般的な投資手法です。

## 2.5 除外すべきターゲットの基準

以下に該当する企業はリストから除外すること:

- ❌ 直近2期連続で赤字の企業
- ❌ 従業員5名以下 かつ 年商5,000万円未満の企業
- ❌ 業種としてワイン・アルコールと相容れない企業（例: アルコール依存症治療施設等）
- ❌ 反社会的勢力チェックに該当する企業
- ❌ 過去に明確な取引拒否の意思表示があった企業
- ❌ 設立1年未満のスタートアップ（資金に余裕がない可能性が高い）

## 2.6 リスト作成の実務手順

### ステップ1: データベースで絞り込み
1. 帝国データバンク or 東京商工リサーチにアクセス
2. 業種コード × 地域 × 売上規模（年商1億円以上を推奨）でフィルタリング
3. 上記30案の業種を中心にリストアップ

### ステップ2: 経営者情報の調査
1. 企業HPで代表者名を確認
2. 可能であれば代表直通番号を調査
3. 経営者のSNS（Facebook、LinkedIn等）でワイン・投資への関心を確認

### ステップ3: 優先度付けとリスト整理
1. 3軸（経済力・親和性・活用動機）でスコアリング
2. Aランクから順に架電リストを作成
3. 1日の架電リストは50〜80件を目安に準備

### ステップ4: リストの鮮度管理
- 3ヶ月ごとにリストを更新
- 架電結果を蓄積し、反応の良い業種・属性を特定
- 月次でターゲット精度の見直しを実施

### タイプ別トーク例

**数字重視型への切り口**:
> 「過去15年間のデータでは、ファインワインは世界株式指数を上回るリターンを記録しています。2024年度のWineBank買付ワインは+6.66%を達成しました。同年のワイン業界平均が-11%ですから、17ポイント以上の差があります。」

**人脈重視型への切り口**:
> 「多くの経営者の方が、重要な接待の場でファインワインをお使いになっています。WineBankなら、接待でお使いにならなかったワインがそのまま資産として値上がりしていく。使っても資産、使わなくても資産、というスキームです。」

**人材重視型への切り口**:

## 3.4 法人としてのワイン活用メリット詳説

### (d) 経営者個人の資産形成

- 法人経由ではなく個人としての投資も可能
- 退職金の運用先としての提案
- 個人のライフスタイル向上

# 第4章 受付突破マニュアル

> **この章の目的**: 社長（意思決定者）に電話をつないでもらうための具体的なテクニックを身につける

## 4.1 受付突破の基本原則5箇条

| 原則 | 内容 | 具体例 |
|------|------|--------|
| **① 営業電話と思わせない** | 「ご提案」「ご確認」「お伝えしたい件」等の言葉を使う | ×「ご案内のお電話です」→ ○「確認したい件がございまして」 |
| **② 受付を味方にする** | 受付担当者を「協力者」として丁寧に接する | 「お忙しいところ恐れ入ります」「お手数おかけします」 |
| **③ 堂々と話す** | 自信のあるトーン、はっきりした声、適切な速度 | 早口にならない、語尾を下げて信頼感を出す |
| **④ 用件に具体性を持たせる** | 曖昧な用件は営業と見抜かれる | ×「ちょっとお話が」→ ○「法人の資産運用に関する最新レポートの件で」 |
| **⑤ 社長の名前を名指し** | 事前に代表者名を調査し、名指しで依頼 | 「△△社長に」と名前を出す → 既知の関係と思わせる |

## 4.2 パターン別突破フレーズ集

### パターン1: 「資料の件」アプローチ（最も効果的）

> **事前準備**: 架電前にDM・メール等で資料を送付しておく

```
「お世話になっております。WineBankの○○と申します。
 先日、△△社長宛にお送りしたお資料の件でお電話いたしました。
 お繋ぎいただけますでしょうか。」
```

**ポイント**: 「送った資料の確認」は営業ではなく事務連絡に聞こえる

### パターン2: 「業界の情報提供」アプローチ

```
「お世話になっております。WineBankの○○と申します。
 △△社長に、○○業界の経営者の方に今注目されている
 法人向け資産運用の最新情報をお伝えしたくお電話いたしました。
 お繋ぎいただけますでしょうか。」
```

**ポイント**: 「業界×情報提供」で専門性と有益性をアピール

### パターン3: 「紹介・ご縁」アプローチ

```
「お世話になっております。WineBankの○○と申します。
 □□様からご紹介をいただきまして、
 △△社長にぜひお時間をいただきたいとのことでお電話しております。」
```

> ⚠️ **実際に紹介がある場合のみ使用**すること。虚偽の紹介は絶対にNG。

### パターン4: 「確認事項」アプローチ

```
「お世話になっております。WineBankの○○と申します。
 △△社長に先日ご案内した件について、
 1点確認させていただきたいことがございまして。
 お繋ぎいただけますでしょうか。」
```

**ポイント**: 「確認」は短時間で済む印象を与え、つなぎやすい

### パターン5: 「伝言依頼」アプローチ（直接つながらない場合のリカバリー）

```
「承知いたしました。
 では恐れ入りますが、WineBankの○○からお電話があった旨、
 △△社長にお伝えいただけますでしょうか。
 社長のご都合のよいお時間に改めてお電話いたしますので、
 ご都合の良い時間帯がもしわかりましたらお教えいただけると助かります。」
```

**ポイント**: 戻り時間の情報を得ることで、次回の架電精度が上がる

## 4.3 受付タイプ別対応戦略

### タイプ①: 秘書タイプ（丁寧にブロック）
- **特徴**: 「どういったご用件でしょうか」と丁寧に聞いてくる
- **対策**: 用件の重要性を具体的に伝える。「法人の資産に関する重要な情報です」等。メモを残してもらう
- **NGワード**: 「営業」「ご提案」（これを聞いた瞬間ブロック）

### タイプ②: ガードマンタイプ（一律拒否）
- **特徴**: 「営業のお電話はお断りしています」と即座に拒否
- **対策**: 時間帯を変えて再架電（早朝・夕方は別の人が出る可能性）。別ルート（問い合わせフォーム、DM）を検討
- **切り返し**: 「かしこまりました。では、△△社長のメールアドレスだけお教えいただけますか。資料をお送りいたします」

### タイプ③: 聞き取りタイプ（用件を詳しく聞く）
- **特徴**: 「どのようなお話ですか？詳しく教えてください」
- **対策**: 簡潔だが魅力的なフックを用意。「法人向けの新しい資産運用サービスで、2024年に+6.66%の実績を出している件です」
- **ポイント**: 受付が社長に伝えたくなるような「数字」や「具体性」を入れる

### タイプ④: 判断委任タイプ（社長に確認してくれる）
- **特徴**: 「少々お待ちください。確認いたします」
- **対策**: 最もチャンスが大きい。用件を明確かつ簡潔に伝える。保留中の待機は粘り強く
- **ポイント**: 受付が社長に伝える「取り次ぎメモ」を意識して、伝わりやすい言葉で用件を伝える

## 4.4 時間帯・曜日別の攻略法

### ゴールデンタイム

| 時間帯 | 特徴 | おすすめ度 |
|--------|------|-----------|
| **8:30 - 9:30** | 始業直後。社長が自席にいる可能性高い。受付が不在の場合も | ★★★★★ |
| **11:30 - 12:00** | 昼休み前。午前の仕事が一段落したタイミング | ★★★★ |
| **13:00 - 13:30** | 昼休み明け。再始動のタイミング | ★★★ |
| **17:00 - 18:00** | 終業間際。受付が帰宅し、社長直通の可能性 | ★★★★ |

### 曜日別のおすすめ

| 曜日 | おすすめ度 | 理由 |
|------|-----------|------|
| 月曜 | ★★ | 週初めの会議・タスク整理で忙しい |
| 火曜 | ★★★★ | 落ち着いて電話に出やすい |
| **水曜** | **★★★★★** | **最も繋がりやすい曜日** |
| **木曜** | **★★★★★** | **水曜と同様に繋がりやすい** |
| 金曜 | ★★★ | 午前は○、午後は外出・早退が多い |

### 中小企業特有の傾向
- 社員数10名以下 → 社長が直接電話に出ることも多い
- 社員数10-50名 → 受付（事務員兼任）がいるが、昼休みや早朝は直通可能
- 社員数50名以上 → 組織化された受付がいる、パターン2-4を駆使

## 4.5 突破できなかった場合のリカバリー策

**リカバリー策は「諦め」ではなく「別ルート開拓」**

| 順位 | リカバリー策 | 具体的アクション |
|------|-------------|----------------|
| 1 | **資料送付の許可を取る** | 「では資料だけでもお送りしてよろしいでしょうか。メールアドレスをお教えいただけますか」 |
| 2 | **社長の戻り時間を確認** | 「何時頃お戻りでしょうか。その時間に改めてお電話いたします」 |
| 3 | **問い合わせフォームからアプローチ** | 第7章の文案を使用してフォーム送信 |
| 4 | **DM（手紙）を送付** | 手書きの一筆箋を添えたDM → 到着後にフォローコール |
| 5 | **別の役職者へアプローチ** | 「財務ご担当の方」「役員の方」へのアプローチに切り替え |
| 6 | **時間帯・曜日を変えて再架電** | 最低3回、異なる時間帯で架電してから判断 |

## 5.2 「よくわからない」の本質と対応戦略

### 「よくわからない」は3パターンに分かれる

#### パターンA: 仕組みがわからない（ワイン投資の概念が新しい）

**対応: 3ステップ簡略説明法**

```
「ご説明が足りず申し訳ございません。
 とてもシンプルな仕組みでして、3ステップです。

 ステップ1: ワインを買います
 ステップ2: 専門の倉庫で保管します
 ステップ3: 値上がりしたらアプリで売ります

 不動産投資のワイン版とお考えください。
 買って、管理して、値上がりを待って売る。
 ただし不動産より遥かに少額から始められます。」
```

#### パターンB: なぜワインなのかがわからない（投資対象としての合理性への疑問）

**対応: 3つの構造的理由を提示**

```
「なぜワインかと申しますと、3つの理由があります。

 1つ目：ワインは生産量が限られていて、増産できません。
 2つ目：飲めばなくなる消費財なので、供給が毎年減ります。
 3つ目：アジアや中東の富裕層の需要が急増しています。

 供給が減って需要が増える。だから価格が上がりやすいんです。
 実際、過去15年で世界株式指数を上回っています。
 世界の富裕層の間では20年以上前から定番の投資先なんです。」
```

#### パターンC: 自分に関係があるかわからない（自分事化できていない）

**対応: 相手の状況に合わせて具体的な活用シーンを提示**

```
（接待が多い経営者の場合）
「△△社長のように取引先との接待が多い方にとっては、
 接待で使うワインがそのまま資産にもなるというメリットがあります。
 使えば接待ツール、使わなければ値上がり資産。
 接待交際費の一部を、消えないお金に変えるイメージです。」

（投資に興味がある経営者の場合）
「△△社長は既に投資をされているとのことですので、
 ポートフォリオの分散先として最適です。
 株や不動産と値動きが連動しないので、
 リスクヘッジの柱として100万円からお持ちいただけます。」
```

## 5.3 心理的ハードル別の対処法

| ハードル | 相手の気持ち | 対処法 |
|----------|-------------|--------|
| **損するのでは** | 元本割れが怖い | 過去15年の実績データ＋保険・補償制度の説明。「仮に値下がりしても飲んで楽しめます」 |
| **管理が面倒** | 手間がかかりそう | 「WineBankが全て代行します。アプリで確認するだけです」 |
| **出口がない** | 売れなかったらどうする | 「マーケットプレイスでいつでも売却可能。コンシェルジュもサポートします」 |
| **周囲に説明できない** | 社内や家族の理解が得られない | 「法人の資産分散として合理的な選択です。実績データを資料でお見せします」 |

## 5.4 NG対応集（絶対にやってはいけないこと）

| NG行為 | 理由 | 正しい対応 |
|--------|------|-----------|
| ❌ 「今買わないと損です」 | 煽り営業は信頼を失う | ○ 「ご都合の良いタイミングでご検討ください」 |
| ❌ 「必ず儲かります」 | 断定的判断は**法律違反** | ○ 「過去の実績では○%です。ただし将来を保証するものではありません」 |
| ❌ 相手の断りを否定する | プライドを傷つけ関係が壊れる | ○ 「おっしゃる通りです。その上で〜」と受け止めてから切り返す |
| ❌ ワイン知識をひけらかす | 経営者のプライドを傷つける | ○ 相手の知識レベルに合わせて平易に説明 |
| ❌ 長時間引き止める | 「しつこい」と思われ二度と取り次いでもらえない | ○ 潔く引き、次回の接点を確保する |
| ❌ 「またお電話します」（日時なし） | フォローが曖昧になる | ○ 「来週○曜日の○時に改めてお電話します」と具体的に |

| メール送付 | **2〜3営業日後** | 届いて目を通す時間を確保しつつ、忘れられる前に |
| 郵送送付 | **5〜7営業日後** | 配送日数＋目を通す時間 |

### 心構え3箇条

1. **「資料は届きましたか」だけで終わらせない** → 必ず次のアクション（アポ設定）に繋げる
2. **1回で取れなくても3回までフォロー** → 間隔をあけて粘り強く
3. **相手が資料を見ていなくても焦らない** → 電話口で要点を伝え、再度の確認依頼＋次回フォロー設定

### フォローコールの回数と間隔

| 回数 | タイミング | 目的 |
|------|-----------|------|
| 1回目 | 送付後2-3日（メール）/ 5-7日（郵送） | 到着確認＋アポ提案 |
| 2回目 | 1回目から1週間後 | 内容確認＋再アポ提案 |
| 3回目 | 2回目から2週間後 | 最終フォロー（新情報を添えて） |

# 第9章 KPI管理・日次運用ガイド

> **この章の目的**: マニュアルの実効性を数字で管理し、継続的に改善する

## 9.1 日次・週次のKPI目標設定

### 日次KPI目標（1人あたり）

| KPI | 目標値 | 計算方法 |
|-----|--------|----------|
| 架電数 | **60〜80件/日** | リスト件数 |
| 受付突破数 | **12〜16件/日** | 架電数 × 受付突破率20% |
| 社長接続数 | **6〜8件/日** | 受付突破数 × 社長在席率50% |
| アポイント取得数 | **1〜2件/日** | 社長接続数 × アポ率15-25% |
| 資料送付数 | **5〜10件/日** | 社長接続数のうちアポ取れなかった先 |

### 週次KPI目標（1人あたり）

| KPI | 目標値 |
|-----|--------|
| 架電数 | 300〜400件/週 |
| アポイント取得数 | 5〜10件/週 |
| 資料送付数 | 25〜50件/週 |
| フォローコール数 | 20〜30件/週（前週の資料送付先） |
| フォローからのアポ取得数 | 2〜5件/週 |

### 重要指標の目安

| 指標 | 目安 | アクション |
|------|------|-----------|
| 受付突破率 | 20%以上 | 20%未満 → 第4章を再確認、フレーズ改善 |
| アポ取得率（対社長接続） | 15%以上 | 15%未満 → 第3章・第5章を再確認、訴求改善 |
| 資料送付率（対社長接続） | 50%以上 | アポが取れない場合でも資料送付に繋げる |
| フォローアポ率 | 10%以上 | 10%未満 → 第8章を再確認、フォロートーク改善 |

## 9.2 架電記録の付け方

### 記録テンプレート

各架電後、以下の項目を必ず記録する:

| 項目 | 記入内容 |
|------|----------|
| 日時 | 20XX年X月X日 XX:XX |
| 会社名 | △△株式会社 |
| 代表者名 | △△ △△ 様 |
| 電話番号 | 03-XXXX-XXXX |
| 業種 | カテゴリ（A/B/C）＋具体的業種 |
| 結果 | 不在/受付NG/社長接続→断り/資料送付/アポ取得 |
| 使用スクリプト | A〜H のいずれか |
| 相手の反応・キーワード | 「忙しい」「興味ない」「面白そう」等 |
| 次回アクション | 再架電（日時）/ フォローコール（日時）/ 対応不要 |
| 備考 | 特記事項（受付の名前、社長の好み等） |

### 記録の活用方法
- **週次ミーティング**で成功パターン・失敗パターンを共有
- **業種別の反応率**を集計し、ターゲット優先度を見直す
- **使用スクリプト別の成約率**を比較し、効果の高いスクリプトを特定

## 9.3 改善PDCAの回し方

### 週次改善サイクル

```
Plan（計画）: 今週のターゲットリスト作成、使用スクリプト決定
  ↓
Do（実行）: 架電実施、記録蓄積
  ↓
Check（確認）: KPI振り返り、成功/失敗パターン分析
  ↓
Act（改善）: スクリプト修正、ターゲット見直し、トーク改善
```

### 週次ミーティングのアジェンダ（30分）

| 時間 | 内容 |
|------|------|
| 5分 | 先週のKPI実績報告 |
| 10分 | 成功事例の共有（どのスクリプト・どのフレーズが効いたか） |
| 10分 | 失敗事例の共有と改善案の検討 |
| 5分 | 今週の目標設定とアクション確認 |

### 改善のチェックポイント

- **受付突破率が低い場合**: 架電時間帯を変える / フレーズを変更する / 事前DMを導入する
- **社長接続後のアポ率が低い場合**: 訴求ポイントを変える / ターゲット業種を見直す / スクリプトを変更する
- **資料送付後のフォロー率が低い場合**: フォロータイミングを調整 / 追加情報の内容を変更
- **特定の業種で反応が良い場合**: その業種へのリスト拡充を優先

## 付録C: 法的注意事項

### 営業活動における遵守事項

| 項目 | 注意点 |
|------|--------|
| **断定的判断の禁止** | 「必ず儲かります」「絶対に値上がりします」は**法律違反**。「過去の実績では〜」と表現する |
| **利益保証の禁止** | 将来のリターンを保証する表現はNG。「過去15年の実績」「2024年度の実績」として事実を伝える |
| **適合性の確認** | 顧客の投資経験・資産状況に合った提案を行う。無理な高額プランの勧誘はNG |
| **クーリングオフ** | 訪問販売に該当する場合、8日間のクーリングオフ制度を説明する義務がある |
| **個人情報の取り扱い** | 架電中に取得した個人情報は適切に管理。目的外利用は禁止 |
| **税務アドバイスの禁止** | 「経費になります」と断定しない。「顧問税理士にご確認ください」と案内する |
| **特定商取引法** | 電話勧誘販売に該当する場合、事業者名・勧誘目的の明示義務あり |

### 電話勧誘における必須告知事項

電話の冒頭で以下を必ず伝えること:
1. 会社名（「WineBankの○○です」）
2. 勧誘目的であること（「サービスのご案内でお電話しています」）
3. サービスの概要（「法人向けのワイン資産運用サービスです」）

> **万が一のトラブル時**: 上長に即報告し、単独で判断しないこと。
